Discount Inn Hotel Management Application

Lead the development of a B2C web portal using Angular, Spring Boot, Spring Data JPA, Maven and MySQL for a Hotel chain. It provides all facilities such as searching hotels, room types, customer booking, billing, staff login using JSON Web Token and Spring Security, booking cancellation, check-in and check-out. The application also sends booking Id and invoice generated in PDF using JasperReports to the customer's email id using a Microservice architecture.
